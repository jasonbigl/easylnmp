LNMP (Nginx/MySQL/PHP) is a web server environment for Linux, supporting CentOS/RHEL/Fedora/Aliyun/Amazon Linux, Debian/Ubuntu/Raspbian/Deepin/Mint Linux VPS or dedicated servers. It can install LNMP, LNMPA (Nginx/MySQL/PHP/Apache), or LAMP (Apache/MySQL/PHP) production environments.

Features include custom Nginx/PHP compile parameters, configurable website and database directories, Let's Encrypt free SSL certificates, HTTP2, TLS 1.3 support, multiple PHP versions, standalone installation options, and utilities like virtual host management, FTP user management, component upgrades, Redis/Xcache installation, MySQL root password reset, 502 auto-restart, log rotation, DenyHosts/Fail2Ban SSH protection, and backup scripts.
